Monster Rolodex 
==============================


## Features
- A signle a page app that displays a collection of Monster Cards.
- Fetches each Card text data and image from API.
- the sreach box filters Crds on Input change 
- Show the filedcards only


## Tech

- [React JS] - HTML enhanced for web apps!
- [Ace Editor] - awesome web-based text editor
- [markdown-it] - Markdown parser done right. Fast and easy to extend.


## Installation
##### Run the following commands to run this project on localhost

```sh
git clone 
```
```sh
cd 
```

(optional) Third:

```sh
npm install
```
